<2> Pixel
 - 일반적인 단위
 - 픽셀이 많으면 상대적으로 크기가 달리질 수 있다
단위
 Dp(Density-Independent Pixel)
 - 화면의 크기가 달라도 동일한 비율로 보여주기 위해서 안드로이드에서 정한 단위
 - 크기가 일정하다


Sp(Scale-Independent Pixel)
 - 시스템 설정에 영향을 받는 단위


DPI(Dot Per inch)
 - 100dp -> 1인치당 픽셀이 100개 들어 있음
